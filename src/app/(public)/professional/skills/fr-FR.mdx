## Développement Back-end

Mon objectif est de créer des solutions robustes, efficaces et évolutives. Pour cela, je travaille actuellement avec les technologies suivantes :

### Langages de programmation

#### Node (TypeScript)

JavaScript a été ma langue de programmation maternelle. Comme c'était mon premier langage, je m'y sens très à l'aise. Oui, il a ses particularités, mais TypeScript est là pour aider.

#### Golang

Mon choix pour le développement back-end. Je l'apprends actuellement, mais je suis déjà conquis. Go est un langage qui capte vraiment mon attention grâce à sa simplicité et ses performances.

### Frameworks & Bibliothèques

#### Express

Un framework d'application web minimal et flexible pour Node.js. Je l'aime car il offre un ensemble robuste de fonctionnalités pour les applications web et mobiles.

### Bases de données

#### MongoDB

Celle avec laquelle j'aime vraiment travailler. C'est une base de données NoSQL utilisant des documents semblables à JSON avec des schémas optionnels.

#### PostgreSQL

En ce qui concerne les bases de données relationnelles, PostgreSQL est mon choix. C'est un puissant système de base de données relationnelle open-source. PostgreSQL a été utilisé pour construire ce site.

#### SQLite

Pour de petits projets et des tests, SQLite est mon choix. Rapide, autonome, fiable et riche en fonctionnalités.

### ORMs & ODMs

#### Prisma

Pour les bases de données SQL et NoSQL, Prisma est mon outil préféré. Il est facile à utiliser, puissant, et offre un client de base de données type-safe. Prisma a été utilisé pour ce site.

#### Mongoose

Parfois, lorsque je n'utilise pas Prisma, j'utilise Mongoose. Il est simple, facile et offre une solution basée sur des schémas.

### Authentification

#### JWT

Pour sa simplicité, j'utilise JWT pour l'authentification. C'est un moyen compact et sûr de représenter des revendications entre deux parties.

#### OAuth2

Pour l'authentification tierce, OAuth2 est mon choix. Simple, sécurisé et largement utilisé pour l'autorisation.

## Développement Front-end

Ma partie préférée du développement. J'adore laisser libre cours à ma créativité et créer des interfaces utilisateur interactives et esthétiques. Voici les technologies que j'utilise actuellement :

### Langages et styles

#### HTML et CSS

Le langage de balisage standard et la feuille de style pour le web. La base de presque toutes les pages et applications web.

#### JavaScript (TypeScript)

Le langage de programmation qui permet des pages web interactives, essentiel pour le développement web moderne.

### Bibliothèques et Frameworks

#### React

React est le plus utilisé pour une raison. C'est une bibliothèque JavaScript pour construire des interfaces utilisateur. Déclaratif, efficace et flexible.

#### Next

Le framework utilisé pour ce site. C'est un framework React qui permet des fonctionnalités comme le rendu côté serveur et la génération de sites statiques.

#### Tailwind

Mon approche préférée pour styliser des applications web, y compris ce site. Il me permet de créer des designs magnifiques et responsives avec un minimum d'effort.

## Outils

En tant que développeur, j'utilise de nombreux outils pour faciliter ma vie. Voici quelques outils que j'utilise quotidiennement :

### Productivité

#### Obsidian

Pour la prise de notes et la gestion des connaissances, j'utilise Obsidian. Une base de connaissances puissante et hautement personnalisable. Je l'utilise pour suivre mes projets, idées et notes.

#### Notion

Notion est mon outil de gestion des tâches. Un espace de travail tout-en-un, mais je l'utilise principalement pour suivre mes routines.

#### Visual Studio Code Insiders

L'éditeur de code que j'utilise pour le développement. Visual Studio Code est assez léger tout en restant puissant. Je l'utilise pour tout ce qui concerne le code.

### Design

#### Figma

Pour concevoir des interfaces utilisateur, j'utilise Figma. Polyvalent, puissant et facile à utiliser, il me permet de créer des wireframes, maquettes et prototypes.

#### Photoshop et Illustrator

Respectivement pour l'édition d'images et de vecteurs. Je les utilise pour créer et éditer des images, illustrations et autres contenus graphiques.

### Systèmes d'exploitation

#### Windows

Mon système d'exploitation principal. Je l'utilise pour tout : développement, design, jeux, etc. Un système d'exploitation polyvalent et puissant.

#### Windows Subsystem for Linux

Pour le développement, j'utilise WSL. Facile à configurer et à utiliser, il me permet d'exécuter une distribution Linux sur Windows. Le meilleur des deux mondes.
