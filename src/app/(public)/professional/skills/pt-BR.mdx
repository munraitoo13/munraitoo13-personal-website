## Desenvolvimento Back-end

Meu objetivo é criar soluções robustas, eficientes e escaláveis. Para isso, estou trabalhando com as seguintes tecnologias:

### Linguagens de Programação

#### Node (TypeScript)

JavaScript foi minha linguagem de programação inicial. Como foi minha primeira linguagem, me sinto muito confortável com ela. Sim, ela tem suas peculiaridades, mas o TypeScript está aqui para ajudar com isso.

#### Golang

Minha escolha para desenvolvimento back-end. Estou aprendendo agora, mas já estou apaixonado. Go é uma linguagem que realmente chamou minha atenção pela sua simplicidade e desempenho.

### Frameworks & Bibliotecas

#### Express

Um framework web minimalista e flexível para Node.js. Gosto dele porque oferece um conjunto robusto de recursos para aplicações web e móveis.

### Bancos de Dados

#### MongoDB

O que eu realmente gosto de trabalhar. É um banco de dados NoSQL que usa documentos no formato JSON, com esquemas opcionais.

#### PostgreSQL

Falando em bancos de dados relacionais, PostgreSQL é minha escolha. É um sistema de banco de dados relacional e objeto-relacional de código aberto e poderoso. PostgreSQL foi usado para construir este site.

#### SQLite

Para projetos pequenos e testes, SQLite é minha escolha. É rápido, autocontido, confiável e um banco de dados completo.

### ORMs & ODMs

#### Prisma

Para bancos de dados SQL e NoSQL, Prisma é minha escolha. Ele é fácil de usar, poderoso e fornece um cliente de banco de dados tipado. Prisma foi utilizado para construir este site.

#### Mongoose

Ocasionalmente, quando não estou usando Prisma, utilizo Mongoose. É simples, fácil e oferece uma solução baseada em esquemas.

### Autenticação

#### JWT

Por sua simplicidade, utilizo JWT para autenticação. É uma maneira compacta e segura de representar reivindicações que serão transferidas entre duas partes.

#### OAuth2

Falando sobre autenticação de terceiros, OAuth2 é minha escolha. É simples, seguro e amplamente usado para autorização.

## Desenvolvimento Front-end

Minha parte favorita do desenvolvimento. Adoro colocar minha criatividade em ação e criar interfaces de usuário bonitas e interativas. Aqui estão as tecnologias com as quais estou trabalhando atualmente:

### Marcação, Estilos & Scripts

#### HTML e CSS

A linguagem de marcação padrão e a linguagem de estilo para a web. São a base de quase todas as páginas e aplicativos na internet.

#### JavaScript (TypeScript)

A linguagem de programação que possibilita páginas web interativas e é uma tecnologia essencial para o desenvolvimento web moderno.

### Bibliotecas & Frameworks

#### React

React é o mais usado por um motivo. É uma biblioteca JavaScript para construção de interfaces de usuário. Declarativa, eficiente e flexível.

#### Next

O framework no qual este site foi construído. É um framework React que permite funcionalidades como renderização do lado do servidor e geração de sites estáticos.

#### Tailwind

Minha abordagem favorita para estilizar aplicações web, incluindo este site. Permite criar designs bonitos e responsivos com pouco esforço.

## Ferramentas

Como desenvolvedor, uso muitas ferramentas para facilitar meu trabalho. Aqui estão algumas das ferramentas que uso no meu dia a dia:

### Produtividade

#### Obsidian

Para anotações e gerenciamento de conhecimento, uso o Obsidian. É uma base de conhecimento poderosa e altamente personalizável. Uso para acompanhar meus projetos, ideias e anotações.

#### Notion

O Notion funciona como minha ferramenta de gerenciamento de tarefas. É um espaço de trabalho tudo-em-um, mas eu o utilizo principalmente para organizar minha rotina.

#### Visual Studio Code Insiders

O editor de código que utilizo para desenvolvimento. O Visual Studio Code é leve, mas poderoso. Eu o uso para praticamente tudo relacionado a codificação.

### Design

#### Figma

Quando estou criando interfaces de usuário, uso o Figma. É versátil, poderoso e fácil de usar, permitindo criar wireframes, mockups e protótipos.

#### Photoshop e Illustrator

Edição de imagens e vetores, respectivamente. Uso-os para criar e editar imagens, ilustrações e outros conteúdos gráficos.

### Sistemas Operacionais

#### Windows

Meu sistema operacional principal. Uso para tudo: desenvolvimento, design, jogos e mais. É um sistema operacional versátil e poderoso.

#### Windows Subsystem for Linux

Para desenvolvimento, uso o WSL. É fácil de configurar e usar, permitindo que eu execute uma distribuição Linux no Windows. O melhor dos dois mundos.
